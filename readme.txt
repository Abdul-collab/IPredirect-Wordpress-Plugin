This is a WordPress plugin designed to provide seamless redirection of users based on their country and page selection. This plugin is particularly useful for delivering localized content, ensuring legal compliance, and optimizing user experience.

== Description ==

This WordPress Plugin is designed so you can easily redirect your user as per the sites Don't miss use this plugin.
Make sure you know how to use this pluign if DON'T this could lead your user to different webpages.

== INSTRUCTION'S / USE ==

1. **Select Page**: Choose the page from the dropdown list that you want to set up redirection for.
2. **Select Country**: Choose the country for which this redirection should apply.
3. **Enter Redirect Link**: Provide the URL to which users from the selected country should be redirected.
4. **Save**: Click the **Create Redirection** button to save the configuration.
5. **Delete**: Delete the record if you want to stop the redirection.



## Features

- **Country-Based Redirection**: Redirect users to different pages based on their country.
- **Page Selection**: Choose specific pages for redirection.
- **Custom Redirect Links**: Set unique redirect links for different country and page combinations.
- **Error Handling**: Alerts for existing records and validation errors.
- **Interactive Admin Interface**: Easy-to-use admin interface with Bootstrap modals and DataTables integration.
- **Deletion of Redirect Records**: Manage and delete existing redirection records directly from the admin dashboard.

== Installation ==

1. **Download the Plugin**:
   - You can download the latest version of the IP-Redirect plugin from the WordPress plugin repository or directly from the source repository.

2. **Upload the Plugin**:
   - Go to your WordPress admin dashboard.
   - Navigate to **Plugins** > **Add New**.
   - Click on **Upload Plugin** and choose the downloaded `.zip` file.
   - Click **Install Now** and then **Activate** the plugin.

3. **Configure the Plugin**:
   - After activation,> **IP-Redirect** in your WordPress admin menu.
   - Use the provided interface to set up redirections based on country and page selection.

## Troubleshooting

- **Duplicate Record Error**: If you encounter an error about duplicate records, ensure that no other redirection exists with the same country and page combination.
- **Database Issues**: Ensure your WordPress installation has the correct database permissions and that the table structure is properly set up.
